# scala-cli-millbuild-crypto-gzip-compress-rsa-encrypted

## Description
Encrypt and decrypt password with RSA.
To compress the encrypted text gzip was used.

## Tech stack
- scala
- millbuild

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
